# How to start
```
npm install
```
## Create your own .env file with:
<ul>
    <li>SERVER_PORT=3000</li>
    <li>SERVER_BASEURL=http://localhost:3000</li>
    <li>DB_USER</li>
    <li>DB_HOST</li>
    <li>DB_NAME</li>
    <li>DB_PASS</li>
    <li>DB_PORT</li>
</ul>

## To run db-migrations, firstly create local PostgreSQL Database,then run command:

```
npm run up
```

## Finally:

```
npm run serve
```
